# Calcul du cercle

Petit projet Python avec interface graphique pour calculer :
- l'aire d'un cercle
- le perimetre d'un cercle

## Bibliotheques

Ce projet utilise seulement les bibliotheques Python standard :
- `tkinter`
- `math`

Il n'y a rien a installer en plus.

## Fichiers

- `main.py` : interface graphique
- `logic.py` : calculs matematiques

## Lancer le projet

Depuis le dossier du projet :

```bash
python main.py
```

Ou avec ton chemin Python :

```powershell
& 'C:/Users/user/AppData/Local/Python/pythoncore-3.14-64/python.exe' .\main.py
```

## Comment ca marche

1. L'utilisateur entre le rayon.
2. Il clique sur `Calculer`.
3. Le programme affiche :
   - l'aire en `cm2`
   - le perimetre en `cm`

## Resume

Projet simple, propre et facile a comprendre.
